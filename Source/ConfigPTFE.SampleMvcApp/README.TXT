
///////////////////////////////////
BEFORE YOU RUN THIS WEB APPLICATION:
///////////////////////////////////

	Run the sample project 'ConfigPTFE.SampleConsoleAppSharedConfigs' before continuing.

	This MVC application's web.config has the configProtectedData entry for 'ConfigPTFE.AdvancedConfigurationProvider'
	set to use an environment.config file on the filesystem for its' initialization data.

	The sample project 'ConfigPTFE.SampleConsoleAppSharedConfigs' will generate the sample config files 
	this MVC app expects to find on your filesystem.
